Job module¶

class Job.Job¶

Bases: object

An abstract class provides a method to specify a goal for a job which will be assigned to a scheduler. Developers are free to create any custom job which overrides goal method and assign to scheduler. This behaviour allows to plug in and out a job to and from scheduler. Please see Scheduler for more details.
