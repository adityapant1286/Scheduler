NotificationService module¶

NotificationService.notify(notifier: notification.notifiers.Notifier.Notifier)¶

This module allows to send a notification based on the custom implementation of the Notifier abstract class. A developer needs to provide implementation for the notifier.

The notifier dependency will be injected by the caller as per the use case. This behaviour allows to plug in either existing notifier or custom notifiers.

This method also captures general audit log.

Parameters:notifier (Notifier) – an instance of custom implementation of Notifier
