APIAuditAgent module¶

class APIAuditAgent.APIAuditAgent¶

Bases: auditlogging.agents.AuditAgent.AuditAgent

Captures the audit trail using a REST endpoint URL (POST) Add this agent to Auditor in order to capture audit log to an endpoint.

Note

  1. If user wants to POST custom JSON request body then,
    pass a valid JSON string to constructor and call Auditor.audit_custom(your_custom_json)
  2. After each call to capture() or capture_custom() latest response is preserved
    until next endpoint request. To get the response, after each invocation please call endpoint_response() to get response
