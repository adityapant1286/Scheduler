Downloader module¶

class Downloader.Downloader¶

Bases: object

An abstract class provides a template for download activities. A subclass must provide implementation for download operation.

Examples

  1. Files download from SFTP or remote location
  2. Files copy from one directory to another

RemoteDownloader module¶

class RemoteDownloader.RemoteDownloader¶

Bases: ioservice.downloaders.Downloader.Downloader

Provides implementation for downloading files from a remote host using SFTP.

__init__()¶

Retrieves required SFTP configurations from configurations

  1. Host
  2. Port
  3. Username
  4. Secret (password or SSH private key path) > this implementation auto detects whether
    to connect using password or private key file
  5. Remote location source directory path
  6. localhost destination directory path
  7. File extensions to be filter

  • dest (str) – destination directory path

Override file extensions for filter.

change_files(files: list = None)¶

Not require for this operation.

Parameters:files – NA
download() → list¶

Connects to SFTP host using configurations and scans for files as per the extensions mentioned in the configurations. If there are any matching files, they will be downloaded to destination with total seconds taken to download files.

Returns:a list of downloaded files ordered by modified time in ascending. Earliest will be the first.
Return type:list
